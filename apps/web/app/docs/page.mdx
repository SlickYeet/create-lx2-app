# TNT-Powered CLI

The TNT-Powered CLI is a powerful tool designed to streamline the creation and
management of TNT and TNT-Powered projects.

## Installation

Install the CLI globally using your preferred package manager:

```bash
# Using Bun
bun add -g tnt-cli

# Using npm
yarn global add tnt-cli

# Using pnpm
pnpm add -g tnt-cli
```

## Usage

Once installed, you can create a new TNT-powered project using:

```bash
tnt create my-app
```

Navigate to the project directory and start the development server:

```bash
cd my-app
bun dev
```

## Commands

Here are some essential commands available in the CLI:

### Create

```bash
tnt create <project-name>
```

Creates a new TNT-powered project.

### Dev

```bash
tnt dev
```

Starts the development server.

### Build

```bash
tnt build
```

Builds the project for production.

## Configuration

You can configure your TNT-powered project using a `tnt.config.ts` file.
Example:

```ts
export default {
  usePayload: true,
  database: "postgresql",
  storage: "s3",
}
```
